## Vilde's Design Style

**Core tension:** Minimal and clean, but with personality. Never bare or cold — always a little warm and a little fun.

### Layout & Space
- Airy and spacious — generous padding, room to breathe
- One focal point per section; don't compete for attention
- Avoid clutter at all costs; if it doesn't earn its place, cut it

### Color
- Muted, earthy palette as the base — desaturated, natural, nothing loud
- Accent colors can be cute and slightly unexpected (dusty rose, sage, warm terracotta, soft lavender) — but keep them soft, not neon
- Use color sparingly; when a color appears, it should mean something

### Typography
- Readability first — clean and legible above all
- Can introduce personality via weight, size contrast, or a single display font if it fits the tone
- Avoid anything that feels corporate or generic

### Personality & Details
- Quirky details are welcome: a subtle pattern, an unusual hover state, a small illustration, a slightly unexpected shape
- Micro-interactions should feel playful and light — not performative or slow
- Copy can have warmth and wit; doesn't need to be dry

### Hard Lines
- Never messy — structure is sacred even when the vibe is playful
- Never tacky — no gradients that scream, no drop shadows for drama, no clashing brights
- No visual noise added "for fun"; every detail should feel intentional
- Animations should be quick and subtle, never distracting

## How to Apply

When making design decisions, default to: *clean structure + muted palette + one small moment of delight*. If something feels too safe, add a quiet detail. If something feels too busy, remove something before adding anything.
